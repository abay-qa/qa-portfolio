# My QA → Developer → AI Learning Roadmap

*Personal learning plan — updated April 2026*

---

## PHASE 1 — Get Hired in QA *(current phase)*

```
✅ Manual Testing
✅ API Testing with Postman
✅ Python & Pytest automation
✅ Performance Testing with k6
✅ Playwright UI Automation
✅ SQL basics
✅ Git & GitHub
✅ Jira bug tracking
⬜ CI/CD basics (GitHub Actions)
⬜ Apply to jobs
```

**Goal:** Land first QA job
**Timeline:** Now

---

## PHASE 2 — Grow on the Job *(while working)*

```
⬜ Python deeper
   - Functions & classes
   - File handling
   - Error handling
   - Reading other people's code

⬜ JavaScript basics
   - Variables, functions, arrays
   - DOM basics
   - Fetch & APIs

⬜ Advanced Playwright
   - Page Object Model pattern
   - Test fixtures
   - Parallel testing

⬜ Advanced Pytest
   - Fixtures
   - Parametrize
   - Conftest
```

**Goal:** Become a strong QA automation engineer
**Timeline:** First 6 months on the job

---

## PHASE 3 — Bridge into Development *(6-12 months in)*

```
⬜ Python for automation
   - Writing scripts that solve real problems
   - Working with files, APIs, databases
   - Building small tools

⬜ Reading and understanding codebases
   - How to navigate a real project
   - Understanding pull requests
   - Writing small bug fixes

⬜ Basic web development
   - HTML & CSS
   - How frontend and backend connect
   - REST API design
```

**Goal:** Fix small bugs yourself, contribute to codebase
**Timeline:** 6-12 months after getting hired

---

## PHASE 4 — AI & Automation *(the real destination)*

```
⬜ AI basics
   - How LLMs work
   - OpenAI / Anthropic APIs
   - Prompt engineering fundamentals

⬜ AI agents
   - Building simple agents in Python
   - Connecting agents to tools and APIs
   - Automating workflows with AI

⬜ QA + AI
   - Using AI to generate test cases
   - AI powered test automation
   - Smart bug detection
```

**Goal:** Build AI powered tools and agents
**Timeline:** Year 2 and beyond

---

## PHASE 5 — Your Own Thing *(the long game)*

```
⬜ Build something of your own
   - A tool, a product, a service
   - Combining QA + AI + fintech knowledge

⬜ Freelance or consulting
   - QA consulting for startups
   - AI automation for small businesses
```

**Goal:** Work on your own terms
**Timeline:** Whenever you're ready

---

## The Thread That Runs Through Everything

```
Python → deeper Python → AI → agents → your own thing
```

Every phase builds on the previous one. Nothing gets wasted.

---

## Key Principles

- Get hired first — you learn 10x faster on a real team
- Don't try to complete all phases before applying
- Each job teaches you what to learn next
- Snowboard principle: fall, figure out what went wrong, go again
